Module 6: Digital Exhibits

Readings

  • Brannock, Jennifer. “Creating an Exhibit in Special Collections and Using it to Promote Collections and Educate Users,” Mississippi Libraries 73(2), 2009. pp. 32-34.
  • Ovadia, Steven. (2014). “Markdown for Librarians and Academics,” Behavioral and Social Sciences Librarian 33, pp. 120-124.
  • “Markdown Syntax” - A cheatsheet for markdown syntax

Installations

  • Text Editor software (comes installed on most computers). Suggestions for text editors:
    • Visual Studio Code is the Microsoft Office equivalent of Notepad or TextEdit. VS Code is more user friendly, and works within your browser (by typing the period . in a github markdown page). For further direction see VS Code’s Installation instructions for MAC or PC users.
    • PC users: Notepad comes already installed
    • Mac users: TextEdit comes already installed
    • If you are using an already-installed text editor, preview your markdown using something like https://dillinger.io/ or https://markdownlivepreview.com/

Deliverables

  • Collaborative Notes Document
  • Final narrative portfolio
